# 5. Error Handling

Provide helpful error messages so users can diagnose issues.

---

## Why error handling matters

Bad: `Error: connection failed`
Good:
```
❌ Unable to connect

Possible causes:
- Network/firewall blocking port 6379
- Service down

Run 'fsf fetch --debug' for details
```

---

## Use `fail()` from `fsf.ui`

```python
from fsf.ui import fail

# Simple
fail("Something went wrong")

# With context
fail("""Unable to connect

Possible causes:
- Network issue
- Firewall

Run 'fsf fetch --debug' for details""")
```

---

## Common error patterns

### Connection errors

```python
except Exception as e:
    err = str(e)
    
    if "10054" in err or "reset" in err.lower():
        fail("""Unable to connect

Possible causes:
- VPN blocking port
- Firewall restrictions

Run 'fsf fetch --debug' for details""")
    
    fail(f"Connection error: {err}")
```

### Authentication errors

```python
if "auth" in err.lower() or "unauthorized" in err.lower():
    fail("""Authentication failed

Run 'fsf connect <provider>' to update credentials""")
```

### Network errors

```python
if "Name or service not known" in err or "[Errno" in err:
    fail("""Unable to resolve host

Check:
- Hostname is correct
- DNS is working

Run 'fsf fetch --debug' for details""")
```

---

## Debug mode

Show full traceback when `debug=True`:

```python
except Exception as e:
    if debug:
        import traceback
        traceback.print_exc()
    fail(f"Error: {e}")
```

## Example: full implementation

```python
def run(config, debug=False):
    from fsf.ui import fail
    
    try:
        # Your fetch logic
        result = fetch_from_storage(config)
        return result
    except Exception as e:
        err = str(e)
        
        # Connection issues
        if "10054" in err or "connection" in err.lower():
            fail("""Unable to connect

Possible causes:
- VPN blocking port
- Firewall

Run 'fsf fetch --debug' for details""")
        
        # Auth issues
        if "auth" in err.lower() or "unauthorized" in err.lower():
            fail("""Authentication failed

Run 'fsf connect <provider>' to update credentials""")
        
        # Default
        fail(f"Error: {err}" if debug else f"Error: {err}")
```
