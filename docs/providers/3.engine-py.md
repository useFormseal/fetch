# 2. engine.py

Implements the fetch logic - retrieves ciphertexts from your storage.

---

## Create engine.py

```python
# fsf/providers/<name>/engine.py
# <name> provider engine

def run(config, debug=False):
    """
    Fetch ciphertexts.

    Args:
        config: Dict with input values from config.json
        debug: Enable verbose error output

    Returns:
        dict[str, bytes]: {id: ciphertext_data}
    """
    # ... fetch logic here ...
    return {"id": b"ciphertext data"}
```

---

## Parameters

| Param | Type | Description |
|-------|------|-------------|
| `config` | dict | Input values from config.json |
| `debug` | bool | Show full traceback? |

---

## Return format

Return a dictionary:

```python
{
    "submission-1": b"formseal.abc123...",
    "submission-2": b"formseal.def456...",
}
```

- **key**: Any unique string (ID, key name, etc.)
- **value**: Ciphertext bytes or string

---

## Minimal example

```python
def run(config, debug=False):
    return {"entry1": b"formseal.ciphertext"}
```
